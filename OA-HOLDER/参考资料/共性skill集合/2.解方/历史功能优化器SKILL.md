# 历史功能优化器 (Historical Feature Optimizer)

本 skill 旨在帮助您通过截图快速将现有功能转化为 HTML 原型，并根据您的反馈进行迭代优化。

## 工作流程

1.  **提供截图**: 您需要提供一个清晰的功能截图。
2.  **原型生成**: 我将分析截图，并将其复刻为一个基础的 HTML 文件 (`prototype.html`)。
3.  **征求反馈**: 创建原型后，我会主动询问您需要进行哪些优化。
4.  **需求描述**: 您可以用自然语言描述您的优化需求（例如：“把这个按钮变大一点，颜色改成蓝色”）。
5.  **原型更新**: 我会根据您的描述更新 `prototype.html` 文件。
6.  **循环迭代**: 您可以继续提供反馈，我会持续更新原型，直到您满意为止。

## 如何开始

请您先准备好一张功能截图，然后对我说：“开始优化历史功能”，我将启动此流程。
