## 快速导航

### 流程概览

```
一句话需求 → 需求澄清 → 用户研究 → 功能规划 → 信息架构 → 原型设计 → 功能清单 + PRD → 交付开发
```

### 核心交付物

- 需求分析文档
- 用户画像与旅程
- 信息架构图
- 流程图
- 原型（低/高保真）
- 功能清单（含优先级）
- PRD

详见 [02-检查项清单](./02-检查项清单.md) 第八节「评审前快速检查清单」。
